.compute_client(config_name = nil, client_id: nil, **options) ⇒ Compute::Client

Create compute client with automatic or specific credential discovery

Examples:

Automatic discovery

client = Eryph.compute_client

Specific configuration

client = Eryph.compute_client('zero')

Specific client ID

client = Eryph.compute_client('myconfig', client_id: 'my-client-id')

With options

client = Eryph.compute_client(logger: my_logger, verify_ssl: false)

Parameters:

  • config_name (String, nil) (defaults to: nil) —

    configuration name for specific config, nil for automatic discovery

  • client_id (String, nil) (defaults to: nil) —

    specific client ID to use

  • options (Hash) —

    additional options

Options Hash (**options):

  • :logger (Logger) —

    logger instance

  • :scopes (Array<String>) —

    OAuth2 scopes

  • :verify_ssl (Boolean) — default: true —

    whether to verify SSL certificates

  • :verify_hostname (Boolean) — default: true —

    whether to verify hostname

  • :ca_file (String) —

    path to CA certificate file

  • :ca_cert (OpenSSL::X509::Certificate) —

    CA certificate object

  • :environment (Environment) —

    environment for dependency injection

Returns:

  • (Compute::Client) —

    a new compute client instance

def compute_client(config_name = nil, client_id: nil, **options)
  # Extract SSL config from options
  ssl_keys = [:verify_ssl, :verify_hostname, :ca_file, :ca_cert]
  ssl_config = options.select { |key, _| ssl_keys.include?(key) }
  
  Compute::Client.new(
    config_name,
    client_id: client_id,
    logger: options[:logger],
    scopes: options[:scopes],
    ssl_config: ssl_config,
    environment: options[:environment]
  )
end
